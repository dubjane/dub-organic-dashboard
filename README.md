# dub Organic Social Dashboard

Editorial-style social media metrics dashboard for dub (TikTok + Instagram).

## Stack
- Next.js 14 App Router
- Tailwind CSS
- Recharts
- TypeScript

## Getting Started

1. Install dependencies:
   npm install

2. Add font files to /public/fonts/:
   - RoslindaleDecNarrow-Semibold.woff2
   - Figtree-Regular.woff2
   - Figtree-Medium.woff2
   - Figtree-SemiBold.woff2
   - Figtree-Bold.woff2

3. Copy .env.example to .env.local and fill in API keys

4. Run development server:
   npm run dev

## Deploy
Deploy to Vercel. All environment variables must be configured in the Vercel dashboard.
